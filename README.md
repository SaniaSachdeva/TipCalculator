Description:
This project is a web-based tip calculator that allows users to calculate tip amounts and split bills efficiently. It focuses on user input handling and responsive interface design.

Tech Stack:
TypeScript
HTML
CSS

Features:
Calculate tip based on bill amount and percentage
Split bill among multiple users
Real-time calculation updates
Responsive user interface

Concepts Applied:
DOM manipulation
Event handling
TypeScript fundamentals

Future Improvements:
Add input validation
Improve UI design
Add currency selection
